FinPlus

O projeto envolveu a construção de uma única página para uma empresa Fintech utilizando HTML, CSS e Bootstrap. Esta tarefa foi desenvolvida para a minha aula de Engenharia de Design Front-End.

Adivinhe o Número Secreto!

API de Voz Responsiva: Integração com a API de Voz Responsiva para adicionar capacidades de feedback por voz.
Estilização Personalizada: CSS personalizado foi aplicado para estilizar os elementos do jogo e layout.
Manipulação de Entrada: Elementos de entrada HTML foram usados para capturar as adivinhações dos usuários e botões para interação.
